In this project we will be developing a web application that Network Architecture Analyzer and Vulnerability Assessment Analyzes a network giving details on the different protocols and services working in the network stack

Services

The webapp read pcap files and analyze a users network and display the network stack

The webapp findes common misconfiguration vulnerabilites that can be found in the network stack.

The webapp is able to generate a layout of the analyzed network and mention some protocols in diffrent layers of the stack
